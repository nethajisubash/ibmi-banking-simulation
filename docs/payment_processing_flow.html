Process Summary

Screening Phase

  • OFAC sanctions check
  • Audit trail creation
  • Status updates

Posting Phase (if approved)

  • Customer account debit
  • General ledger entries
  • Status change to 'Posted'
  • SWIFT message queuing

Technical Implementation Notes

  • Each procedure call includes error handling and rollback capabilities
  • Audit records maintain complete transaction trail
  • Status field progression: Pending → Approved/Rejected → Posted
  • SWIFT messages only generated for successfully posted transactions
